# SWIFT Codes JSON

This repository provides a structured list of **SWIFT (BIC) codes** for banks and financial institutions around the world, stored in a single JSON file: `swift_codes.json`.

## Overview

The file contains verified SWIFT codes along with corresponding bank details, country, and branch information. It is designed to help developers and organizations integrate reliable financial institution data into their systems, such as for payment gateways, fintech platforms, or compliance tools.

## File Structure

Each record in `swift_codes.json` follows this format:

```json
{
    "country": "VIETNAM",
    "name": "HONG LEONG BANK VIETNAM LIMITED",
    "city": "HANOI",
    "branch": "HANOI BRANCH",
    "swift": "HLBBVNVX002"
}
